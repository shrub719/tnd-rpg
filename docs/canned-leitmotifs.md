# Canned Leitmotifs

The DM's Guide provides a few examples of leitmotifs, but it can be difficult to balance
them in combat. This is my best attempt at providing a list of tried-and-tested, game-ready
leitmotifs. The list outlines the core mechanics (requirements and effects of the motif),
but not any flavour or theming so the leitmotifs can hopefully be moulded to many
characters.

Remember the requirements and effects are *examples* of well thought-out, balanced leitmotifs,
and the names are placeholders. Make the leitmotifs your own!

Name | Requirements | Effects
---- | ------------ | -------
Special move | Using a certain weapon. | Can range from a modifier to a guaranteed hit.
Traveral | Make a check. | Allow for more movement than the DM would normally in combat, e.g. via a teleport.
Duo attack set | Proximity to target. | Set the next person up for a duo attack.
Duo attack | Must be popcorned from an ally who set their attack. | Three chances at making an attack roll.
Key change | Uninterrupted by attack. | Switch stances; choose a new key for the encounter.
