# Canned Leitmotifs

The DM's Guide provides a few examples of leitmotifs, but it can be difficult to balance
them in combat. This is my best attempt at providing a list of tried-and-tested, game-ready
leitmotifs. The list outlines the core mechanics (requirements and effects of the motif),
but not any flavour or theming so the leitmotifs can hopefully be moulded to many
characters.
