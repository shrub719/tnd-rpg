# The Note Desolation

TND is a simple TTRPG system intended to be easy to pick up for players new to the genre,
while still enabling creativity in roleplay and combat.
It was made specifically for my own campaign and setting, and heavily (and transparently)
inspired by Disco Elysium's own system.
